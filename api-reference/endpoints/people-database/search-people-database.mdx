## Request body

You may use the following parameters:

<ParamField query="filters" path="param" type="array" required>
  Filters you want to use in your query
</ParamField>

<ParamField query="page" path="param" type="number">
  Page you want to filter on (optional)
</ParamField>

<ParamField query="size" path="param" type="number">
  Number of results asked, between 1 and 100 (optional)
</ParamField>

<ParamField query="excludes" path="param" type="array">
  Array of properties to exclude from the results (optional)
</ParamField>

### Filters

Each filter should have the following properties:

<ParamField query="filter id" path="param" type="string" required>
  Filter identifier
</ParamField>

<ParamField query="in" path="param" type="array">
  List of the values you want to include
</ParamField>

<ParamField query="out" path="param" type="array">
  List of the values you want to exclude
</ParamField>

<Note>
  To know which filters you are able to use, refer to the [GET Filters](/api-reference/endpoints/people-database/get-database-filters) section.
</Note>

Example

```json
{
	"filters": [
		{
			"filterId": "country",
			"in": ["France"],
			"out": []
		},
        {
            "filterId": "currentCompany",
            "in": ["Lemlist"],
            "out": ["Lemwarm"]
        }
	],
	"page": 1,
	"size": 25,
	"excludes": ["education", "skills"]
}
```

</Note>
